# useCoreApi

`useCoreApi` is an hook used to to perform api request base on `@commercelayer/sdk` and `swr`.

Example:

<Source
  language="tsx"
  dark
  code={`
const {
  data: order,
  isLoading,
  mutate: mutateOrder,
  isValidating
} = useCoreApi(
  'orders',
  'retrieve',
  [
    id,
    {
      include: [
        'market',
        'customer',
        'line_items',
        'shipping_address',
        'billing_address',
        'shipments',
        'payment_method',
        'payment_source'
      ]
    }
  ],
  {
    isPaused: () => isMockedId(id),
    fallbackData: makeOrder()
  }
)
`} />
